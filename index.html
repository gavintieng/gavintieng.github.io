Projects

Estimating Player Impact in the 2016-17 NBA Season

Using NBA player salary, Twitter engagement, and Wikipedia pageviews, we attempted to create a model that accurately predicted a given player's impact estimation (PIE) score. Four different statistical methods were used, including linear regression, logistic regression, decision trees, and random forests.

Predicting Yelp Ratings Using Inspection Data

Using Toronto Public Health Inspection Data (DineSafe), we created a predictive model that utilizes Random Forest Classifiers and Logistic Regression to accurately predict the Yelp rating of a given establishment. Constructed using SQL, PySpark, MLLib, and JuypterLab.

Using Time Series Analysis to Forecast Rising Temperatures

Using data scraped from the Australian Bureau of Meteorology, I conducted a time series analysis to construct a SARIMA model that accurately forecasted future temperatures in the city of Melboure, Australia.

Using Regression Analysis to Predict the Number of Active Doctors

This project aimed to investigate the effect of 14 different variables on the mean number of active physicians in 1990. We aimed to find a model that both utilized the most relevant factors and analyzed the relationship between predictors.

Visualizing Uber Ride Data

A simple project that uses R to visualize rideshare data from Uber, including the number of trips every hour and every month.

World Happiness Analysis

Using Python and Jupyter Notebook, I created a visualization of some of the variables that may predict happiness levels around the world.
